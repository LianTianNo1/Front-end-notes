    # 前言

    算法为王。

    想学好前端，先练好内功，内功不行，就算招式练的再花哨，终究成不了高手；只有内功深厚者，前端之路才会走得更远。

    强烈推荐 GitHub 上值得前端学习的数据结构与算法项目，包含 gif 图的演示过程与视频讲解。

    # GitHub 项目

    # 数据结构与算法

    关于数据结构与算法的 GitHub 项目，star 数由高到低排序。

    • javascript-algorithms（51.3k） (opens new window) 该仓库包含了多种基于 JavaScript 的算法与数据结构，提供进一步阅读的解释和链接。 每种算法和数据结构都有自己的 README，包含相关说明和链接，以便进一步阅读 (还有 YouTube 视频) 。
    • Algorithm_Interview_Notes-Chinese（24.6k） (opens new window) 2018/2019/校招/春招/秋招/算法/机器学习(Machine Learning)/深度学习(Deep Learning)/自然语言处理(NLP)/C/C++/Python/面试笔记
    • algorithm-visualizer（23k） (opens new window) 算法可视化工具是一个交互式的在线平台，可以从代码中可视化算法。
    • swift-algorithm-club（20.8k） (opens new window) 算法和数据结构迅速，有解释 ！
    • algorithms（15.4k） (opens new window) Python中数据结构和算法的最小示例。
    • algo（8.3k） (opens new window) 数据结构和算法必知必会的50个代码实现。
    • awesome-algorithms（6.4k） (opens new window) 一份很棒的学习和/或练习算法的地方的整理清单。
    • algorithms (opens new window) 对Jeff的算法书、笔记等进行错误跟踪
    • javascript-algorithms（5.4k） (opens new window) 此存储库包含不同著名计算机科学算法的 javascript 实现。
    • Learn-Algorithms（3.8k） (opens new window) 算法学习笔记。 AlgoXY（3.8k） (opens new window) 基本算法和数据结构手册。
    • JS-Sorting-Algorithm（2.9k） (opens new window) 一本关于排序算法的 GitBook 在线书籍 《十大经典排序算法》，使用 JavaScript & Python & Go & Java 实现。
